# Harsha Pavan - Personal Portfolio

Welcome to the repository for my personal portfolio website! I am an **AI/ML Engineer** and **Python Developer** passionate about building intelligent systems, training deep learning models, and turning complex data into actionable insights.

## 🚀 Live Preview
Check out the live portfolio: [https://harsha-2005.github.io/Portfolio/](https://harsha-2005.github.io/Portfolio/)

## ✨ Features
- **Modern & Responsive Design:** Fully responsive layout built with Tailwind CSS that looks perfectly polished on mobile, tablet, and desktop monitors.
- **Dark/Light Mode:** Seamless manual toggling between dark and light themes with smooth transitions.
- **Interactive Hero Section:** Features a dynamic looping typewriter effect for professional titles and a beautifully glowing profile image.
- **Project Showcase:** Dedicated section highlighting featured projects with GitHub repository links and interactive, responsive technology tags.
- **Certifications & Achievements:** Visual, image-first layouts showcasing professional certifications, internship completion certificates, and competitive hackathon placements.
- **Functional Contact Form:** A fully working, asynchronous contact form integrated with [Web3Forms](https://web3forms.com/).

## 🧰 Tech Stack
- **HTML5:** Semantic document structure
- **Tailwind CSS:** Utilizing the JIT (Just-In-Time) CDN compiler for rapid UI development, custom color thematic mapping, and beautiful hover states
- **Vanilla JavaScript:** Powers the typewriter hero effect, robust project filtering, dark mode switching, and form submission validation
- **FontAwesome:** Beautiful scalable vector icons

## 💻 Local Execution
To run this project locally on your machine, simply clone the repository and open the `index.html` file in your preferred web browser. No local development server or npm installation is strictly required!

```bash
git clone https://github.com/Harsha-2005/Portfolio.git
cd Portfolio
```
*(Then simply double-click the `index.html` file to open it in Chrome, Edge, Safari, etc.)*

## 🌐 Deployment Settings
This static site is optimized to be deployed completely free on platforms like **GitHub Pages**, **Netlify**, or **Vercel**. 
1. If using **GitHub Pages**, navigate to your repository `Settings > Pages`, set the source branch to `main`, and save. It will natively host your site.
2. If using **Netlify/Vercel**, you can simply drag and drop the root folder into their deployment dashboard.
